# NuGet Alternatives to SunamoDebugging

This document lists popular NuGet packages that provide similar functionality to SunamoDebugging.

## Overview

General debugging utilities

## Alternative Packages

### System.Diagnostics
- **NuGet**: System.Diagnostics.Debug
- **Purpose**: Built-in debugging tools
- **Key Features**: Debug, Trace, Debugger class, assertions

### Serilog
- **NuGet**: Serilog
- **Purpose**: Diagnostic logging
- **Key Features**: Structured logging, context enrichment

### Ben.Demystifier
- **NuGet**: Ben.Demystifier
- **Purpose**: Better stack traces
- **Key Features**: Readable async stack traces, source line info

### MiniProfiler
- **NuGet**: MiniProfiler
- **Purpose**: Performance profiling
- **Key Features**: Web request profiling, database queries

## Comparison Notes

Built-in System.Diagnostics for basic debugging. Ben.Demystifier for better error analysis.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Diagnostics**: Built-in debugging tools
- **Serilog**: Diagnostic logging
- **Ben.Demystifier**: Better stack traces
